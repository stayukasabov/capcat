**PEP 701 multi-line f-string** (`article_fetcher.py:2138`):
```python

f"Fallback system downloaded {
    len(downloaded_images)} additional images"

# After (Python 3.9+):
f"Fallback system downloaded {len(downloaded_images)} additional images"
```

**Runtime union type annotation** (`interactive.py:759`):
```python
# Before (Python 3.10+ only at runtime):
def _find_latest_index_html() -> str | None:

# After (Python 3.9+ safe):
def _find_latest_index_html() -> "str | None":
```

**Diagnostic method**: `_discover_custom_sources` swallows all exceptions. Added a temporary test `test_hn_source_loads_directly` that re-raised the exception, exposing the real SyntaxError on CI. Removed after root cause confirmed.
