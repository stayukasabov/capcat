### B23 Fix — CLI space-separated source names ignored
- `capcat fetch bbc guardian` only fetched bbc; `guardian` silently dropped
- Root cause: `cli.py:476` used `args[0].split(",")` — only read first positional arg
- Fix: `[s.strip() for a in args for s in a.split(",") if s.strip()]` — all args collected
- Error message updated to say "space- or comma-separated"
- Merged, installed via `pipx install . --force`

### PDF drain silent freeze fix (v1.9.19)
- `wait_until_idle` was polling silently for up to 120s while PDF downloads were active
- Fix: logs `"Downloading PDFs: N active, M queued, K completed"` every 10s during drain
- Test: `tests/unit/test_pdf_drain_progress.py` — verifies INFO log emitted while active, not emitted when idle
- First iteration used `caplog` — failed order-dependently due to closed logging stream from background thread; replaced with `patch.object(manager.logger, "info")`

### PDF note placement fix (v1.9.20)
- Note ("Downloading PDFs can slow down...") was appearing after the PDF yes/no choice due to questionary terminal rendering reordering it
- Fix: moved note into the static SUMMARY block (printed before questionary renders anything)

### TUI exit terminal clear fix (v1.9.21)
- Exit printed "Exiting interactive mode." and left TUI output visible
- Fix: both exit points (`action == 'exit'` and `choice == 'exit'`) now call `os.system("clear")` before returning/exiting
