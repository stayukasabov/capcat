## Root Causes / Key Findings

**B6 — Source YAMLs absent after fresh init**

`init_project` created `Config/sources/active/{config_driven/configs,custom,bundles}` as empty directories. `SourceConfigMirror.is_mirrored()` checks only for directory existence, so it returned `True` on the first fetch. This caused `UnifiedSourceProcessor` to call `check_for_upgrades()` instead of `run_first_mirror()`. With no manifest and no files, `_resync_manifest()` ran and produced an empty manifest. No YAMLs were ever copied.

Fix: removed the dir creation from `init_project`. `SourceConfigMirror._mirror_*()` methods already call `dst.mkdir(parents=True, exist_ok=True)` — they own the dirs. `source_registry.py` uses `.exists()` guards throughout and handles absent dirs gracefully.
