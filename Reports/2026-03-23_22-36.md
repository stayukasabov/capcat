### InfoQ Image Embedding Bug — Root Cause Found and Fixed
- Comprehensive instrumented debugging traced the full pipeline from HTML fetch → image extraction → download → URL replacement
- Root cause: `_extract_image_urls` in `capcat/core/image_processor.py` was using `data-src` attribute before `src`
- On InfoQ pages: `src` = full CDN URL (`https://imgopt.infoq.com/fit-in/3000x4000/filters:quality(85)/...`), `data-src` = relative lazy-loading path (`news/2026/03/.../Kleppmann 1.jpg`)
- The extractor picked `data-src` → `urljoin(base_url, relative_path)` doubled the path → 404 download failures → `url_mapping` had no article body image keys → `replace_image_urls` found no matches → markdown kept remote URLs
- The formatter (`_process_images`) correctly reads `src` first, so the markdown got the right CDN URL — creating a permanent key mismatch
- Fix: Changed priority from `srcset > data-src > src` to `srcset > src > data-src` in `_extract_image_urls`, aligning it with the formatter's actual attribute priority

## Root Causes / Key Findings

**InfoQ image embedding failure:**
InfoQ HTML structure:
```html
<img src="https://imgopt.infoq.com/fit-in/3000x4000/filters:quality(85)/..."
     data-src="news/2026/03/.../Kleppmann 1.jpg">
```
- `_extract_image_urls` used `data-src` first → wrong relative URL → `urljoin` doubled path → 404
- Formatter used `src` first → correct CDN URL → markdown had right URL
- `url_mapping` keys never matched markdown URLs → no replacement possible

**Secondary finding:** `UnifiedMediaProcessor._load_source_config` uses lowercase relative path `sources/active/config_driven/configs/iq.yaml` but the actual user config lives at `Config/sources/active/config_driven/configs/iq.yaml` (capital `Config`). This caused InfoQ to always use default `["img"]` selector instead of the configured `["article img", "figure img", ...]` selectors. With the default selector, sidebar thumbnails (not present in article markdown) were also downloaded. This secondary issue does not break functionality but causes unnecessary downloads of unrelated images.
