Sources Package


Overview


This package contains the following modules:


  • sources.__init__
  • sources.specialized.__init__ - Specialized source implementations for platforms like Medium and Substack
  • sources.active.__init__
  • sources.base.config_schema - Base configuration schema for news sources
  • sources.base.__init__ - Base classes and interfaces for the source system
  • sources.base.factory - Clean factory pattern for creating news source adapters
  • sources.active.custom.lb.source - Lobsters source implementation for the new source system
  • sources.active.custom.hn.source - Hacker News source implementation for the new source system
  • sources.specialized.twitter.__init__ - Twitter/X
  • sources.specialized.twitter.source - Twitter/X
  • sources.specialized.substack.source - Substack
  • sources.specialized.vimeo.__init__ - Vimeo specialized source
  • sources.specialized.vimeo.source - Vimeo specialized source implementation
  • sources.specialized.medium.source - Medium
  • sources.specialized.youtube.__init__ - YouTube specialized source
  • sources.specialized.youtube.source - YouTube specialized source implementation
