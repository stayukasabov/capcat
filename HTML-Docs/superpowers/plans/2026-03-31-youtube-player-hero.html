YouTube Player Hero Implementation Plan


> For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.


Goal: Replace the CLI demo block in docs/index.html with a Plyr-based custom YouTube player.
Architecture: Swap the static <pre class="code-demo"> block inside .hero-demo with a Plyr YouTube embed div. Add Plyr CSS to <head>, add Plyr JS before </body>, initialize in main.js. Update .hero-demo CSS to remove padding/background that fights the video — use an aspect-ratio wrapper so the player is responsive at 16:9. No new JS files; no npm; CDN only.
Tech Stack: HTML, CSS, vanilla JS, Plyr 3.7.8 (CDN)

File Structure


| File | Change |

|------|--------|

| docs/index.html | Add Plyr CSS to <head>; replace hero-demo inner block; add Plyr JS before </body> |

| docs/css/main.css | Update .hero-demo — remove padding/bg, add aspect-ratio wrapper; remove .code-demo rule |

| docs/js/main.js | Add Plyr initialization |

Task 2: Replace hero-demo inner block


Files:
  • Modify: docs/index.html (hero section)

The block to replace is approximately lines 103–114:


<span id="demo-heading" class="visually-hidden">Example command-line usage</span>
<div class="hero-demo">
  <pre class="code-demo" role="region" aria-labelledby="demo-heading"><code class="language-bash">$ ./capcat bundle tech --count 30
Fetching from 3 sources in bundle 'tech'
◯ STARTING HACKER NEWS (30 ITEMS)
◯ STARTING LOBSTERS (30 ITEMS)
◯ STARTING IEEE SPECTRUM (30 ITEMS)
✓ 90 articles saved successfully
Saved to: ../News/news_2025-11-24/</code></pre>
</div>

  • [ ] Step 3: Replace with Plyr embed

Replace the entire block above with:


<div class="hero-demo">
  <div class="hero-demo__player"
       data-plyr-provider="youtube"
       data-plyr-embed-id="dQw4w9WgXcQ">
  </div>
</div>

The id="demo-heading" span and aria-labelledby are no longer needed — the player has its own accessible title from YouTube. Remove them.

Task 4: Add Plyr JS and initialize


Files:
  • Modify: docs/index.html (before </body>)
  • Modify: docs/js/main.js

  • [ ] Step 6: Add Plyr script tag

In docs/index.html, after the existing script tags (jQuery, GSAP, main.js) and before </body>, add:


<script src="https://cdn.plyr.io/3.7.8/plyr.js"></script>

The order must be: jQuery → GSAP → main.js → plyr.js (Plyr has no jQuery dependency; order relative to main.js doesn't matter).


  • [ ] Step 7: Initialize Plyr in main.js

In docs/js/main.js, append at the end of the file:


// YouTube player
if (document.querySelector('[data-plyr-provider="youtube"]')) {
    new Plyr('[data-plyr-provider="youtube"]', {
        controls: ['play-large', 'play', 'progress', 'current-time', 'mute', 'volume', 'fullscreen'],
        youtube: { noCookie: true, rel: 0, showinfo: 0, modestbranding: 1 }
    });
}

  • [ ] Step 8: Test player in browser

Open docs/index.html locally. The Plyr player should render with custom controls. Click play — the YouTube video should load (requires internet). Confirm no JS console errors.
