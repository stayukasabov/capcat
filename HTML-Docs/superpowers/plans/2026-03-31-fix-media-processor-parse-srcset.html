Fix MediaProcessor.parse_srcset CDN Comma Bug Implementation Plan


> For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.


Goal: Fix MediaProcessor.parse_srcset so it correctly handles CDN proxy URLs (Substack, Cloudinary) that embed comma-separated transformation parameters in the path, enabling Substack images to be downloaded and embedded in HTML output.
Architecture: formatter.py::_parse_srcset was fixed in v1.6.7 with a descriptor-aware fragment-merging algorithm. MediaProcessor.parse_srcset is a duplicate implementation that still uses naive srcset.split(","). The fix ports the equivalent algorithm into parse_srcset. The only intentional divergence from formatter.py: the x-descriptor float check is inlined as try/except float() instead of delegating to the module-level _is_float helper (which lives in formatter.py and should not be imported across modules for a private helper). Functionally identical. No interface changes — same method signature and return type.
Tech Stack: Python 3, BeautifulSoup4, pytest

Context for the implementer


The broken URL pattern (observed in CLI debug output)

DEBUG: Failed to download https://senykamara.substack.com/p/fl_progressive:steep/https%3A%2F%2F...

The URL fragment fl_progressive:steep/... is the tail of a Substack CDN path after a comma split corrupted it. The real URL is:

https://substackcdn.com/image/fetch/$s_X,w_1456,c_limit,f_auto,fl_progressive:steep/https%3A%2F%2F...

Naive srcset.split(",") splits on every comma, including the ones inside w_1456,c_limit,f_auto,fl_progressive:steep.


The correct algorithm (already in formatter.py::_parse_srcset, lines 221–288)

A srcset entry ends when its last space-separated token is a valid descriptor (Nw or Nx). Until that descriptor appears, any comma-split fragment is a continuation of the previous URL and must be re-joined.   • [ ] Step 4: Replace parse_srcset body

Replace the entire body of MediaProcessor.parse_srcset (from if not srcset: through return "" at line 750) with:


if not srcset:
            return ""

        # Reject pure data: URI placeholders before any comma-splitting
        if srcset.strip().startswith("data:"):
            return ""

        def _ends_with_descriptor(s: str) -> bool:
            tokens = s.rsplit(" ", 1)
            if len(tokens) != 2:
                return False
            desc = tokens[1].strip()
            if desc.endswith("w") and desc[:-1].isdigit():
                return True
            if desc.endswith("x"):
                try:
                    float(desc[:-1])
                    return True
                except ValueError:
                    pass
            return False

        # Merge comma-split fragments that are continuations of the same URL.
        # A fragment is a continuation when it does not start a new URL scheme
        # AND the previous accumulated entry has not yet ended with a descriptor.
        raw_parts = srcset.split(",")
        entries: list = []
        for part in raw_parts:
            stripped = part.strip()
            starts_new = stripped.startswith(("http://", "https://", "//", "data:"))
            if not entries or starts_new or _ends_with_descriptor(entries[-1]):
                entries.append(stripped)
            else:
                entries[-1] += "," + part

        best_url = ""
        max_width = 0

        for entry in entries:
            parts = entry.rsplit(" ", 1)
            if len(parts) == 2:
                url, descriptor = parts
                url = url.strip()
                if url.startswith("data:"):
                    continue
                descriptor = descriptor.strip()

                if descriptor.endswith("w"):
                    try:
                        width = int(descriptor[:-1])
                        if width > max_width:
                            max_width = width
                            best_url = url
                    except ValueError:
                        continue
                elif descriptor.endswith("x"):
                    try:
                        float(descriptor[:-1])
                        if not best_url or max_width == 0:
                            best_url = url
                    except ValueError:
                        continue
            elif len(parts) == 1:
                url = parts[0].strip()
                if url.startswith("data:"):
                    continue
                if not best_url:
                    best_url = url

        return best_url

Keep the docstring unchanged — only the body changes.
