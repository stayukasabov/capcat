# Sources Package

## Overview

This package contains the following modules:

- [`sources.base.config_schema`](./config_schema.md) - Base configuration schema for news sources
- [`sources.base.factory`](./factory.md) - Clean factory pattern for creating news source adapters
- [`sources.active.custom.lb.source`](./source.md) - Lobsters source implementation for the new source system
- [`sources.active.custom.hn.source`](./source.md) - Hacker News source implementation for the new source system
- [`sources.specialized.twitter.source`](./source.md) - Twitter/X
- [`sources.specialized.substack.source`](./source.md) - Substack
- [`sources.specialized.vimeo.source`](./source.md) - Vimeo specialized source implementation
- [`sources.specialized.medium.source`](./source.md) - Medium
- [`sources.specialized.youtube.source`](./source.md) - YouTube specialized source implementation
