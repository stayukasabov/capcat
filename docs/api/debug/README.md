# Debug Package

## Overview

This package contains the following modules:

- [`debug.fix-sources`](./fix-sources.md) - Source Fix Tool
Automatically suggests and applies fixes for problematic sources
- [`debug.simple-troubleshoot`](./simple-troubleshoot.md) - Simple troubleshoot script without emojis per CLAUDE
- [`debug.content-extraction-debugger`](./content-extraction-debugger.md) - Content Extraction Debugging Tool for Capcat
Helps diagnose why articles return empty markdown or fail to fetch
- [`debug.source-tester`](./source-tester.md) - Source Configuration Tester
Tests Capcat source configurations and suggests fixes
- [`debug.log-analyzer`](./log-analyzer.md) - Capcat Log Analyzer
Analyzes Capcat logs to identify patterns in failures and warnings
- [`debug.quick-troubleshoot`](./quick-troubleshoot.md) - Quick Troubleshoot Script
Immediately diagnose the three failing URLs from the error messages
