# Htmlgen Package

## Overview

This package contains the following modules:

- [`htmlgen.lb.generator`](./generator.md) - Lobsters specific HTML generator implementation
- [`htmlgen.hn.generator`](./generator.md) - Hacker News specific HTML generator implementation
- [`htmlgen.base.base_generator`](./base_generator.md) - Base HTML Generator for Compartmentalized HTML Generation System
- [`htmlgen.lesswrong.generator`](./generator.md) - LessWrong specific HTML generator implementation
