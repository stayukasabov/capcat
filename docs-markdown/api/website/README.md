# Website Package

## Overview

This package contains the following modules:

- [`website.apply_design_system`](./apply_design_system.md) - Apply design system CSS and JS to all HTML files in the docs folder
- [`website.apply_full_layout`](./apply_full_layout.md) - Apply full layout (header, footer, back-to-top button, container wrapper) to all docs HTML files
- [`website.fix_nav_links`](./fix_nav_links.md) - Fix navigation links in all docs HTML files to match root index
